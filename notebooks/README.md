# Notebooks

This folder contains Jupyter notebooks related to the Airline Flight Delay Prediction project.

Current notebooks:

- **EDA_Airline_Delay.ipynb**  
  Performs exploratory data analysis including:
  - Dataset inspection
  - Missing value analysis
  - Distribution analysis of arrival delays
  - Seasonality trends
  - Weather impact analysis
  - Creation of classification target variable

Future notebooks will include regression modeling, classification modeling, and time-series forecasting.
